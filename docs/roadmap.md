# Product Roadmap

## High priority after the internal MVP proves the workflow

- **AI-powered copy generation with Claude or OpenAI API:** improve variety and client-specific
  quality after the guided workflow is validated.
- **Proper PDF export:** improve client handoff with a consistent server-rendered, jsPDF, or
  html2canvas export.
- **Client-shareable kit links:** add when the product moves beyond localStorage and needs secure
  cloud-backed sharing.
- **Automatic hosted design-request tracking and inbox sync:** add only after the manual JSON import
  and Web3Forms request workflows prove demand during MVP validation.

## Medium priority

- **LinkedIn section:** useful for professional local services after the core channel mix is stable.
- **TikTok/Reels section:** useful for video-first businesses after the core content workflow is
  stable.

## Nice to have

- **30-day content calendar:** add after the core kit flow is stable and monthly-pack demand is
  proven.
